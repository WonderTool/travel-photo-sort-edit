# Code Of Conduct

This project should be practical, kind, and privacy-conscious.

## Standards

Expected behavior:

- Be respectful and specific in feedback.
- Assume contributors are trying to help.
- Keep critique focused on the workflow, docs, or code.
- Protect privacy when discussing photo examples or user data.
- Avoid sharing identifying images, metadata, locations, or private trip details without permission.

Unacceptable behavior:

- Harassment, insults, or targeted personal attacks.
- Publishing private images or personal information.
- Pressuring users into destructive photo cleanup.
- Dismissing safety concerns around originals, backups, or private media.

## Reporting

If you see a problem, open a GitHub issue with the minimum detail needed to explain it. Do not include private photos, secrets, personal contact details, or sensitive metadata in public reports.
